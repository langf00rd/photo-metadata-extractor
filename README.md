# Photo Metadata Viewer

Minimal Notion-style EXIF metadata viewer — Next.js 16, TypeScript, Tailwind CSS.

Drop a photo → extracts full EXIF → displays in a clean card with toggleable fields.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel (recommended)

```bash
npm i -g vercel && vercel
```

Or push to GitHub and import at vercel.com/new — zero config.

## Deploy to Netlify

```bash
npm run build && netlify deploy --prod
```
